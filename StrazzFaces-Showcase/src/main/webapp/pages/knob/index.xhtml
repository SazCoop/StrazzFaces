Knob

s:knob it's a wrapper of this jQuery plugin by Anthony Terrien. It's useful to insert numeric values in a range.

Ajax

An ajax listener fired on the "change" event

Max,Min and Step

You can set the max and min value (the default range is 0-100), it's also possible to set a custom increment/decrement step.

Show/Hide label

you can hide the value label

Label template

just like PrimeFaces's ProgressBar, you can set the output format of the label

Disabled

like any other input component the knob can be disabled

Cursor

setting this attribute to true the bar is not filled, leaving just a little cursor to show the value of the knob

Thickness

you can change the thickness of the bar

Colors

you can change the color of your knob. You can use an hex value, a css constant or a java.awt.Color object

@ManagedBean
@RequestScoped
public class KnobBasicBean implements Serializable {

	private static final long serialVersionUID = 1L;

	private int value = 25;

	public void confirm() {
		FacesContext.getCurrentInstance().addMessage(null, new FacesMessage("You chose " + value));
	}

	public int getValue() {
		return value;
	}

	public void setValue(int value) {
		this.value = value;
	}
}
